新增單字

💡 Claude 會生成詳細翻譯、例句、語感說明(約 NT$0.30–0.90)

💡 用你自己的 Claude 查單字,把 JSON 回應貼回來。完全不消耗 API 費用,適合 Claude Max 訂閱者。
🔁 Claude 對話有記憶,**第一次** 給過 prompt 後,後續直接輸入單字即可,**不用再產生 prompt**。

點上面「產生 Prompt」會根據目前輸入的單字與語言更新內容。

💡 適合單字書批次錄入。日後可在卡片裡按「請 Claude 詳細解釋」升級。

這批新增的單字都會自動標上這些分類。

單字 詞性 意思

💡 同單字寫多列 = 多個詞性/意思,送出時會合併成一筆。
💡 意思欄用「;」分不同意思(會自動編號 ①②),「、」分同義詞。
